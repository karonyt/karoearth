条項ごとの詳細
言葉の定義について
・「発言」とはチャット・看板・キルログなどでの媒体を主に指します。
・「表現」については、建築やマップアート、スキンによる表現を主に指します。
・ 1ブロック = 1m と換算します。

第１条
運営及び管理者の指示・裁定に従うこと

運営とは ADMIN や MODERATOR などのことを指します。運営からの警告、改善命令を受けた場合それに従う必要があります。運営の中でも最高権限を持っているのはサーバー主の かろん であり、必要に応じて既存の処罰内容が変更されることもあります。

第２条
建国規約に同意すること

建国規約はMinecraft内規約の下に記載されています。

第３条
他人を不快にさせる発言や表現、ネガティブキャンペーンの禁止

他人が不快になるものとして該当するものは主に
・挑発
・差別
・スパム (連投行為)
・R18コンテンツ（暴力・出血・犯罪・性器・排泄）
・本人の同意の有無に関わらず個人情報全般
これらが主に処罰対象となります。身内同士でしか通じない話題についても、不特定多数が閲覧可能状態である場合、スパム（連投行為）と判定されることがあります。/tellコマンドなどを使用したいわゆるDM機能でも処罰されることがあります。機種による差別も処罰対象となります。
処罰 : 永続MUTE もしくは 即刻のBAN

【処罰例】
・「かろEarthくそすぎわろす」と発言 → 即刻のBAN
・「Unfair運営、サーバーやめちまえ」と発言 → 即刻のBAN
・「〇〇くんは××市にすんでる男の子だよ」→ 即刻のBAN
・「死ね」と発言→ 即刻のBAN
・「〇〇ばか」と発言→ 即刻のBAN
・「うせろ」と発言→ 即刻のBAN
・流血描写、性器を露出したスキンでの参加 → 即刻のBAN

第４条
宗教に関係する表現の禁止

宗教に関してですが、宗教と文化は結びつきが強いため一部に於いては寛容的です。
【許容されているもの】
・寺社仏閣やモスクなどの宗教建築
・スキンにおける宗教の関連する服装
これら(カルト宗教については例外)については処罰されることはありません。

【処罰されるもの】
・宗教を露骨にテーマとした発言及び表現
・カルト宗教をモチーフとした発言及び表現
処罰 : 即刻のBAN

第５条
原則処罰に対する異議申し立ては受け付けない

処罰 : 永続MUTE

第６条
ウイルスなどの有害なファイル、コンテンツを添付しないこと

ウイルスやマルウェア、クラッシュGIF等のURLもこれに含まれます。

処罰 : 即刻のBAN

第７条
基本的に個人間の問題、言い争い等は当事者同士で解決すること

基本的に規約に違反しない限りは個人間の問題には介入しません。サーバーの風紀を乱す行為については、特に問題となっているプレイヤーを処罰する可能性があります。

第８条
不具合・グリッチ・バグの使用禁止

処罰基準の目安は以下の通りです。

処罰の目安

【他人の財産を侵害する、経済に悪影響を与えるもの】
・本来戦争できない地域に侵攻して破壊や略奪、家畜を倒すなどした
・アイテムを無限増殖した
・管理者コマンドを実行してレベルを不正に上げた
・本来想定されていないであろう稼ぎ方を使用した

（過去に他鯖であった例）
・AdminShopで購入したブロックを石切り台で切り出して加工、その後売却し、その差額で儲けた
・玄武岩生成装置を作成し、スピード・効率強化のエフェクトを利用して周回装置を作った

※ 自分の稼ぎ方がグリッチ行為であるか心配な場合には運営にご相談ください。
※ 他人に知られたらまずいというような稼ぎ方はアウトである可能性が高いです。

【他のプレイヤーとの間に不公平な優位性が発生するもの】
・本来併用不可能なエンチャントをつける
・本来片手持ちを想定したアイテムで両手持ちでも機能が発揮されてしまった場合
※ 悪質性によって処罰が変わります（悪質性に関する基準は非公開）

【軽度】基本処罰なし、場合によっては警告を受けるレベル
定義 : あまり他人に迷惑をかけない、気にならないレベルのもの
・本来は入れない区域にエンダーパール等を使って入る行為
・ラグを利用して地下を透視する行為
※ 軽度なバグを利用して破壊行為や窃盗、家畜を倒す等した場合,BANされる可能性があります。

第９条
大規模地形改変の禁止

ここは整地サーバーではないため大規模な地形破壊については処罰を行っております。
【大規模と捉えられる地形破壊行為の例】
・現実で世界遺産に登録されている山を原型の無い形にまでに整地する
・幅20m以上ある水域の水ブロックを埋めたてる
・海上での建築（大小関わらず）（浮かんでいるものも含める）

【過去に他鯖であった例】
・琵琶湖の水を抜いた
・富士山を噴火させた
・ヒマラヤ山脈を平野にした
・ユーラシア大陸を水没させた
・北海道を沈没させた
・佐渡島を沈めた
・南極大陸の雪を全てマグマブロックに変えて情熱大陸にした
・死海の水を抜いてマグマで埋め、本当の意味の死海にした
・世界中の海や川の水を全てスポンジで消した
・太平洋にアトランティス大陸を出現させた
・建築などの目的なしで、3チャンク丸石で領土を覆った
・四国を水没させた
・ガンジス川の上をすべてブロックで覆った
・太平洋の真ん中にコンビナート施設を作成した
・海上に天空都市を造営した

【許可される事例】
・戦艦や豪華客船の建造
（※1⃣ 全長100m以内）（※2⃣ 1プレイヤーにつき、6隻まで建造可能）（※3⃣ 正方形に近い物は撤去されることがある）
・水深5m以上を確保した海中での建築
・架橋などの通路（縦350m横40m以内）（陸上に到達する度、制限はリセットされます）
→ 小さな島を経由して350m以上の通路を作ることは可能
・整地などの護岸工事（元の海岸線から15m以内の改変に収まるように努めること）

処罰 : 違法建築の撤去
処罰 : あまりに超大規模な場合 : 即刻のBAN

第１０条
チートやハック・テクスチャの使用禁止

例外として以下のものは使用可能です。
【かろEarthで認められているてテクスチャの一覧】
・ブロックの見た目を変更するだけのテクスチャ（透過ができるものを除く）
・RTXテクスチャやシェーダー
・チャンク表示テクスチャ

処罰 : 即刻のBAN

【クライアントについて】
全て禁止です。

第１１条
サブアカウントの使用禁止

家族兄弟などの同居している人がいる等の理由であれば大丈夫です。外出先のため、一時的にアカウントを切り替える等も認められています。サブアカウントでログインボーナスを常習的に受け取っていた場合などには処罰される可能性があります。
処罰 : 各ケースによる

第１２条
サーバーに負荷をかける行為の禁止

故意に読み込みチャンクを増やしたり、無意味なパケットを送信するなど対象は様々です。

【サーバーに負荷をかける行為の例】
・モブを大量に湧かす行為
・チェストなど負荷が掛かるブロックの大量設置
・サーバーに負荷を掛ける装置の作成
・ボートやトロッコ、防具立ての大量召喚
・Crusher 等のチート
処罰 : 各ケースによる

第１３条
マクロ、等自動化ツールの禁止

自動釣り機の作成や自動採掘機、外部ツールを使ったAI搭載のBOTなども処罰対象です。トロッコに乗りながら右クリックだけで収穫ができる装置も場合によっては処罰対象となります。

【禁止されているものの例】
・自動釣り機
・自動採掘機
・スポナートラップ
・モブを1か所に集める装置
・モブを倒す装置
・マウス、ボタン等に重りを乗せて放置(手を使わないもの)
・Minecraftを見ずに半自動的に手動で採掘、建築等を行う行為
・モブトラップ
・玄武岩生成装置

【許可されるものの例】
・サトウキビ自動収穫機
・水流式エレベーター
・クラフターを用いた自動クラフト装置
・丸石製造機、石製造機、焼き石製造機
・自動かまど
・自動仕分け装置
・サボテン自動収穫機

処罰 : 即刻のBAN

第１４条
荒らし・窃盗・他者の移動の自由を制限する行為の禁止

<荒らし行為に該当する例>
・ラバキャッスル
・領土内建築物の破壊
・他国の領土周辺を荒らす行為
・荒野にブロックを不規則に置き景観を損なわせる行為

処罰 : 即刻のBAN

<窃盗>
窃盗を目的とした入国等
※ 基本的には、信頼できない人物を入国させない・個人でのチェスト保護を心がけてください。
※ 事実関係の把握・判断が難しいため、対処されない可能性が高いことをご承知おきください。
処罰 : 各ケースによる

<他者の移動の自由を制限する行為>
ここで定義されていることは物理的な移動の自由についてです。スポーン地点を黒曜石などのブロックで囲う等行為、または、落下させてダメージを与えるようなものも禁止されています。
処罰 : 即刻のBAN

第１５条
商売での詐欺行為禁止

アイテムを偽造する行為は禁止です。詐欺に関しても 第１７条 運営では原則アイテムやお金の補填を行わない
の通り補填は原則行いません。運営で追跡し、没収できたアイテムに関しては返却される可能性があります。また、窃盗に関与、共犯した国や個人、組織は同じく処罰を受ける可能性があります。
処罰 : ひどいものは即刻のBAN

第１６条
不適切なスキンの使用を禁止

いわゆる透明スキンと呼ばれるようなものやR18コンテンツ等の不適切なスキンの使用は禁止です。
処罰 : 即刻のBAN

第１７条
サーバーの方針に同意できない人は処罰されることがある

【処罰例】
・「OOが強すぎる弱体化すべき」とMinecraft内チャットでしつこく発言
・「酒を追加しろおおおお」とMinecraft内チャットでしつこく発言
・「平和主義はヌルゲー」などと批判意見をMinecraft内チャットでしつこく発言
処罰 : 永続のMUTE、酷い場合には即刻のBAN

第１８条
運営では原則アイテムやお金、プレイヤーステータスの補填を行わない

アイテムをロールバックなどで補填すると稀にアイテムが増殖してしまうためです。なるべく保護チェストを活用したり、大切なものはエンダーチェストにしまうようにお願いします。不具合は必ずと言っていいほど発生し、バグでもアイテムやお金が失われることがあります。その際も大変心苦しいのですが補填は一切行えないことを承知でプレイしてください。

第１９条
創作された表現物について かろEarthは責任を負わず、作成者が責任を負うこと

マップアートについて作者はそれを表現した責任を負う必要があります。他者の著作物を使用する際にはくれぐれもお気をつけください。

第２０条
サーバーメンテナンス中に何度もログインしない

作業中にログが連投されると見えにくくなるからです。
処罰 : ひどいものは即刻のBAN

第２１条
その他、運営が処罰が必要と判断した者は処罰される可能性がある

上記規約に該当しない場合でも運営が処罰が必要と判断した場合、BANされる可能性があります。予めご了承ください。

かろEarth 建国の規約

かろEarthでは建国のシステムとしてMakeCountryというアドオンを使用しています

サーバーで皆様が快適に遊ぶためにも以下の決まりに同意してもらう必要があります

第1条 総則
この規約は変更される場合があります
また運営により処罰や違反基準が変更される場合もあります

第2条 名称及び説明について
国や国際組織、ロールの名称及び説明を以下のような名にすることを禁じます
・許可されていない個人情報の名称
・実際に存在する宗教団体や政治団体の名称
・社会的モラルに反した名称
・差別、誹謗中傷などを含んだ名称
・その他運営が不適切と判断した非常識な名称

処罰：国名の改称、従わない場合国の消去

第3条 領土について
第1項
全て海域であるチャンクの領土取得の禁止

※ただし海岸から12チャンクは領有可能 第2項 他国を妨害する取り方の禁止
以下の場合処罰されます
東西南北のうち三方向以上他国の広げる余地をなくした場合
線上領土(目安として幅3チャンク以下、長さがその4倍以上のもの)
第3項 囲い込み領土の禁止
国境の外縁のみをbuychunkしたり、buychunkしたエリアに空白の穴を作ったりして、中をあたかもbuychunkしたようにする行為は発見され次第処罰されます

処罰：領土の抹消、国の消去

第4条 国王の義務と禁止事項
国王は以下の事を守る義務があります
①国民に対してサーバーのルールを伝える義務
②運営から改善命令が出された際には国家を変革する義務

処罰: BAN、後継者がいない場合国家解体

分かりやすい規約
建国の規約を分かりやすく要約しました。
ただし基準等は上の本規約に準じます。仮にこれだけを読んで規約違反に問われても自己責任とします。

No.1
この規約はあくまで目安で、最終判断は運営が下します

No.2
個人情報とか誹謗中傷のような、他人が嫌な気持ちになる国名は禁止です

No.3
①海の領有は海岸から12チャンク以外は禁止です
②棒のような領土を作ったり領土で他国を妨害したりするのは禁止です
③周りだけを領有して中身を塗ったように見せるのは禁止です

No.4
国王は運営の勧告に従い、規約を国民に知らせる義務があります

処罰について
【ほとんどの場合】
・国王がBANされる
・国家解体
・問題となっている領土の消去
